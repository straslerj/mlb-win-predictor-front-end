About

This project is designed to attempt to predict the winner of baseball games (or prove that doing so is not really possible...) using machine learning. To learn more about this effort, please visit the GitHub repo for this project to view source code, an architecture diagram, and further explanations.

The point of this website is simply to visualize the progress and provide some insights into the logic behind the machine learning part. The details of the current state of the model can be seen in the below table.

Disclaimer: this is really a preliminary model -- I wasn't actually planning to try and predict anything until the 2024 season and was going to build the model in the offseason. Why am I saying this? Mostly, I don't want you to judge me for my model's lackluster performance, but I also hope you keep coming back to see if there have been any improvements.
